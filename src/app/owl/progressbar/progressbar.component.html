Description

Owl Progress Bar is an Angular component for indicating progress and activity.

Import

Getting Start

Owl Progress Bar requires a value [value] between 0 and 100 to display the progress. User could also provide a buffer value via property [bufferValue] to indicate how much the buffer has been done. 


]]>

Appearance

Owl Progress Bar has 3 different types of appearance: 'basic', 'tradition' and '3d'. User could set the appearance via property [type]. 




]]>

Properties of owl-progressbar

Name Type Optional Default Description
bufferValue number Optional 0 Defines the buffer value of the progress bar. Max: 100, Min: 0.
value number Optional 0 Defines the value of the progress bar. Max: 100, Min: 0.
hideValue boolean Optional false When specify it true, it would hide the value label.
type 'basic' | 'tradition' | '3d' Optional 'basic' Defines the appearance of the progress bar. Valid options: 'basic', 'tradition' and '3d'.
string string Optional '%' Defines the unit of the progress bar value
