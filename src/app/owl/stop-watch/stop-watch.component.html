Description

Owl Stop Watch is an Angular component to measure the amount of time elapsed from a particular time when it is activated to the time when the piece is deactivated. It also like the traditional stop watch allows user to record split times or lap times.

Import

Getting Start

Use owl-stop-watch into your component. It has three button controls to toggle the timer, record lap time and reset the timer. 


]]>

Appearance

By default, Owl Stop Watch has a traditional stop watch 'clock' appearance. There are two: 'analog' and 'digital'. User could set it via property [type]. 



// 'digital' appearance

]]>

Lap Item

When user clicks on the lap button, a list of lap item returns from event (watchLap). Here are the properties of a OwlStopWatchLapItem:

  • order: number -- the order of the lap item in the lap list
  • time: number -- the span of time in milliseconds from the start of the watch to the lap.
  • timeFormatted: string -- the span of time in format from the start of the watch to the lap.
  • timeDiff: number -- the span of time in milliseconds from the last lap to this lap.
  • timeDiffFormatted: string -- the span of time in format from the last lap to this lap.
  • timeTotalDiff: number -- the span of time in milliseconds from the first lap to this lap.
  • timeTotalDiffFormatted: string -- the span of time in format from the first lap to this lap.

Keyboard Interaction

  • ENTER -- Toggle the Stop Watch
  • SPACE -- Record the lap times
  • ESC -- Reset the Stop Watch

Properties of owl-stop-watch

Name Type Optional Default Description
type 'analog' | 'digital' Optional 'analog' Defines the type of stop watch appearance.
showControlPanel boolean Optional true Defines whether to show the control panel of the stop watch
disabled boolean Optional false Defines whether the stop watch is disable.
tabIndex number Optional null Index of the element in tabbing order

Events of owl-stop-watch

Name Parameters Description
watchStart null Event emits when the stop-watch gets started.
watchPause null Event emits when the stop-watch gets paused.
watchReset null Event emits when the stop-watch gets reset.
watchLap originalEvent: event
lapList: OwlStopWatchLapItem[]
 Event emits when the stop-watch gets lapped.

Functions of OwlStopWatchComponent

OwlStopWatchComponent is exported as owlStopWatch. Here are the public functions from it:

Name Parameters Return Description
toggle event: any void A toggle action to start or pause the stop-watch
start event: any void start the stop-watch
pause event: any void pause the stop-watch
reset event: any void stop the stop-watch and reset the stop-watch value to 0.
lap event: any void take a lap on the running stop-watch
