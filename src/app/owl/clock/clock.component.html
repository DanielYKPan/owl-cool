Description

Owl Clock is an Angular component to show a clock-like timer.

Import

Getting Start

Use ]]> into your component. As Owl Clock is design responsively, you could modify the clock size via setting the font-size of ]]>. 


]]>

Owl Clock also provides three different types of appearance('analog' -- default, 'flipbox' and 'digital'). You could change it via property [type]. 




]]>

Clock in Different Timezone

Owl Clock could also be used in different timezone. Set property [gmtOffset] to a GMT Offset to show clock in your desire locations. For example: Sydney -> 10, Beijing -> 8 , New York -> -5, Berlin -> 1

Note: once you set an offset, it's fixed and won't change on its own (i.e there are no DST rules). If you want an actual time zone, you might have to use [startTime] to set an initial time of your desire timezone to start the clock. 




]]>

Properties owl-clock

Name Type Optional Default Description
type 'analog' | 'flipbox' | 'digital' Optional 'analog' Defines the type of clock appearance
gmtOffset number Optional null Set the GMT Offset to show a clock in different timezone. Default is to show local time.
locale
Optional
Defines the locale value of the clock
showCharacters boolean Optional false Defines whether to show clock number on the clock analog.
showMeridian boolean Optional false Defines whether to show time meridian in clock analog.
startTime Date Optional null Defines the time that the clock starts at.

Events of owl-clock

Name Parameters Description
clockTick date: Date -- the now moment. Event emitted when the clock tick.
