Description

Angular BackToTop button is a Angular button component to allow you instantly scroll back to top. It is customisable and animated.

Import

Getting Start 


]]>

Bind to Specific Element

By default, the BackToTop button would scroll back to top of the window. To customise this, set property [bindToElm] to the HTML element you wish to scroll back to its top.

Customise Button

By default, the BackToTop has a arrow-up icon inside. To customise this, provide any html content within owl-back-top-content and put them inside owl-back-top. 


    
        
    

]]>

The BackToTop button would appear when scrollY distance is 200px by default. User can override property [showOnDistance] to number value. If it is set to 0, the BackToTop button would always show on the page. 



]]>

By default, the BackToTop has a fix position on the bottom-right corner of the page. To customise this, provide a desirable position setting through [ngStyle] on owl-back-top. 


    

]]>

Animation

By default, animation is applied when scrolling back to top. Set property [speed] in order to slow down or speed up the scroll speed. The default speed is 80 which means it would scroll 80px in 15ms. If you want the scrolling more dramatic, set property [acceleration]to a number value. This is a coefficient and it would increase or decrease the speed value in every 15ms. 


]]>

In addition, set property [animate] to false to disable the animation, this would let you scroll back to top instantly once you click the button. 


]]>

Dependencies

None

Properties of owl-back-top

Name Type Optional Default Description
aria-label string Optional "Scroll Back to Top" Used to set the 'aria-label' attribute for a meaningful label.
acceleration number Optional 0 Acceleration coefficient, added to speed when using animated scroll
animate boolean Optional true When specified to false, the scrolling process would NOT be animated.
bindToElm HTMLElement Optional null A specific HTML element that user intend to scroll back to its top.
disabled boolean Optional false When specified to true, the BackToTop button would be disabled.
rippleEffectDisabled boolean Optional false When specified to true, the ripple effect on the button would be disabled.
showOnDistance number Optional 200 The BackToTop button would appear on the page when scrollY is on this distance. If it is set to 0, the button would always appear on the page.
speed number Optional 80 The speed of the scrolling process.

Events of owl-back-top

Name Parameters Description
reachTop null Emits when scroll to top process is done.
