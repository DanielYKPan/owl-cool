Description

Owl FanMenu is an Angular button to show a bunch of fan wings as menu items. The FanMenu button is movable and draggable around the screen and the FanMenu is spinnable.

Import

Getting Start

Owl FanMenu consists of several FanMenuWing owl-fan-menu-wing. Users need to provide their menu content inside owl-fan-menu-wing, and use those wings inside owl-fan-menu.

Wing Color & Wing Angles

Each owl-fan-menu-wing has color 'red' as their background color. User could use property [wingColor] to override the default wing background color. In addition, each wing takes a '30'deg of the whole fan menu circle. User could use [wingAngles] to override the default value.

Drag and Spin

By default, the menu is draggable and movable. When the movement stops, the menu would recover to its nearest corner of the screen. Set property [movable] to false and the menu would have a solid position. In addition, the fan menu is NOT spinale, set property [spinale] to true to override this behaviour.

Properties of owl-fan-menu

Name Type Optional Default Description
closeOnSelected boolean Optional false When specified to true, the fan menu would be close when one the fan men wing is selected.
gutterLeft number Optional 30 The space between the menu and the left boundary of the page window. Value unit is 'px'.
gutterTop number Optional 30 The space between the menu and the top boundary of the page window. Value unit is 'px'.
gutterRight number Optional 30 The space between the menu and the right boundary of the page window. Value unit is 'px'.
gutterBottom number Optional 30 The space between the menu and the bottom boundary of the page window. Value unit is 'px'.
radius number Optional 200 Specify the size of the menu wings from the center of the button. Value unit is 'px'.
btnOffset number Optional 25 Specify the gap between the menu button and the menu item wings. Value unit is 'px'.
btnRadius number Optional 56 Specify the size of the menu button. Value unit is 'px'.
btnText string Optional 'Menu' The text inside the menu button.
btnHtml string Optional null Specify the inner html content of the menu button. This value would override the value of btnText
wingAngles number Optional 30 Specify the angles of each wings. Value unit is 'deg'.
menuPosition string Optional 'topLeft' Specify the start position of menu button. Valid value includes: 'topLeft', 'topRight', 'bottomLeft' and 'bottomRight'.
movable boolean Optional true Specify whether the menu is movable.
spinable boolean Optional false Specify whether the menu is spinable.
startAngles object Optional
The angle when the menu open at four different positions and before it expands. The object has four properties topLeft, topRight, bottomRight and bottomLeft with values to be defined in Number. The default value 'topLeft: 0' means the menu would open on 3 o'clock degree when the menu is in topLeft position. (topRight: 90 -> 6 o'clock, bottomRight: 180 -> 9 o'clock, bottomLeft: 270 -> 12 o'clock)

Properties of owl-fan-menu-wing

Name Type Optional Default Description
wingColor string Optional 'red' Specify the background color of the wing
disabled boolean Optional false When specified to true, the wing is not selectable.

Events of owl-fan-menu

Name Parameters Description
menuExpanded null Emits when menu is expanded.
menuCollapsed null Emits when menu is closed.

Events of owl-fan-menu-wing

Name Parameters Description
wingSelected null Emits when menu wing is clicked.
