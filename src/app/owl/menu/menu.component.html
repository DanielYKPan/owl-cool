Description

Owl Menu is an Angular component normally for your apps navigation. It would create a overlay panel with a list of options. It is well styled and animated.

Import

Getting Start

owl-menu requires a list of options( owl-menu-item) inside it. By itself, owl-menu does not open an overlay panel. It needs to be attached to a menu-open trigger via [owlMenuTriggerFor].

Open and Close menu Programmatically

User could open or close the menu panel programmatically via open() / close() from owlMenuTriggerFor directive.

Customise Menu Position

By default, the menu panel will be rendered below(y-position), after(x-position), and overlapping its trigger. The position can be changed using the property xPosition(‘before’ | ‘after') and the property yPosition(‘above’ | ‘below’). The menu can also be set to not overlap the trigger using property [overlapTrigger]=‘false’.

Nested Menu

Owl Menu supports the ability of opening a sub-menu via owl-menu-item. In the case, users have to create their root menu and sub-menus, and use [owlMenuTriggerFor] on owl-menu-item that should trigger the sub-menu.

Keyboard Interaction

Shortcut Action
DOWN_ARROW Move focus to next menu item.
UP_ARROW Move focus to previous menu item.
RIGHT_ARROW Open a sub-menu if it is the trigger
LEFT_ARROW Close a sub-menu
ENTER Open the menu panel
ESC Close all menu panels

Properties of owl-menu

Name Type Optional Default Description
overlapTrigger boolean Optional true Defines whether the menu panel should overlap the trigger
xPosition 'after' | 'before' Optional 'after' Defines the position of the menu panel in the X axis relative to its trigger.
yPosition 'above' | 'below' Optional 'below' Defines the position of the menu panel in the Y axis relative to its trigger.

Events of owl-menu

Name Parameters Description
close null Event emits when menu panel is about to close.
closed null Event emits when menu panel is closed.

Properties of owl-menu-item

Name Type Optional Default Description
disabled boolean Optional false Defines whether the menu item should be disabled.
rippleEffectDisable boolean Optional false Defines whether the ripple effect should be disabled.

Properties of owl-menu-trigger-for

Name Type Optional Default Description
owlMenuTriggerFor OwlMenuComponent Optional null References the menu instance that the trigger is associated with
disabled boolean Optional false Defines whether the menu trigger should be disabled.

Events of owl-menu-trigger-for

Name Parameters Description
menuOpened null Event emitted when the associated menu is opened.
menuClosed null Event emitted when the associated menu is closed.
