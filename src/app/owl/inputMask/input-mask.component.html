Description

Owl Input Mask is an Angular directive used with HTML input element. It ensure input value in a certain format such as numeric, date, currency, email, phone and serial number.

Import

Getting Start

owlInputMask is used with HTML input element with a regex mask string. It is compatible with @angular/forms and supports both FormsModule and ReactiveFormsModule. This means Angular Form directives ngModel and formControl can be used to bind to your input value. 


]]>

Mask

Mask format can be a combination of the the following built-in definitions

  • a - Alpha character (default: A-Z,a-z)
  • 9 - Numeric character (0-9)
  • * - Alpha numberic character (A-Z,a-z,0-9)

]]>

Users can also define their own regex pattern for alpha character 


]]>

Underscore(_) is the default placeholder for a mask and this can be customized using [owlInputMaskSlotChar] option. 


]]>

If the input does not complete the mask definition, it is cleared by default. Use [owlInputMaskAutoClear] property to control this behavior. In addition, certain part of a mask can be made optional by using ? symbol where anything after the question mark becomes optional. 


]]>

Properties of owlInputMask

Name Type Optional Default Description
owlInputMask string require null Defines the mask pattern.
owlInputMaskAutoClear boolean Optional true When specified to false, the directive would clear the incomplete value on blur.
owlInputMaskCharacterPattern string Optional '[A-Za-z]' Defines the regex pattern for alpha characters
disabled boolean Optional false When specified to true, the input would be disabled.
readonly boolean Optional false When specified to true, the input would be set to readonly.
owlInputMaskSlotChar string optional '_' Defines the placeholder character in mask, default is underscore.
owlInputMaskUnMask boolean optional false Defines if ngModel sets the raw unmasked value to bound value or the formatted mask value.
