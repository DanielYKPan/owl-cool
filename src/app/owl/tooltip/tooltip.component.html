Description

Owl Tooltip is an Angular directive that provides a text label for a host element. The tooltip would be displayed with styles and animations when user hovers over or longpresses an element.

Import

Getting Start

Owl Tooltip is applied to an element with owlTooltip directive where the value of the directive defines the text to display.

Tooltip Position

The tooltip will be displayed below the element by default and this can be configured using [owlTooltipPosition]. There are four choices: 'below'(default), 'above', 'left' and 'right'.

Tooltip Trigger

Owl Tooltip gets displayed on hover event of its target by default, other option is the focus event to display and blur to hide. This can be configured via [owlTooltipTrigger]

Tooltip Delay

Owl Tooltip is displayed or hidden instantly by default. However, user may add delays via [owlTooltipShowDelay] and [owlTooltipHideDelay]. They accept a number value in terms of milliseconds.

Manually calling showTooltip() and hideTooltip()

To manually cause the tooltip to show or hide, you can call the showTooltip() and hideTooltip() directive methods, which both accept a number in milliseconds to delay before applying the display change.

Tooltip Global Config

Tooltip default config could be set by providing options via the OWL_TOOLTIP_DEFAULT_CONFIG injection token. In this case, every tooltip in your app would have that default config. Beware, user could still override individual tooltip's config via tooltip directive API.

Dependencies

None

Properties for owlTooltip

Name Type Optional Default Description
owlTooltip string required null Defines the content of the tooltip.
owlTooltipTrigger 'hover' | 'focus' optional 'hover' Defines the trigger event to show the tooltip.
owlTooltipDisabled boolean optional false When specified to true, the tooltip would be disabled.
owlTooltipEscape boolean optional true When specified to false, it supports HTML tags in the content.
owlTooltipPosition 'left' | 'top' | 'right' | 'bottom' optional 'bottom' Defines the position of the tooltip.
owlTooltipHideDelay number optional 0 Delay to hide the tooltip in milliseconds.
owlTooltipShowDelay number optional 0 Delay to show the tooltip in milliseconds.
owlTooltipSticky boolean optional false When specified to true, the tooltip would stick forever once it appears.
owlTooltipTouchendHideDelay number optional 1500 Delay to hide the tooltip in milliseconds in touch device.
