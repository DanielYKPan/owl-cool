Description

Owl Sweet Alert provides an Angular service OwlSweetAlertService to open a overlay window with styles and animations. It's built as Angular version of the original SweetAlert.

Note: this is a dialog with limited configure options. If you want to have a more flexible and customisable dialog (such as configuring the dialog appearance or its functions), you may consider to use OwlDialog.

Import

Getting Start

SweetAlert dialog is open via using OwlSweetAlertService#swal() function. Users need to provide a config options to pass data to SweetAlert.

The OwlSweetAlertService#swal() function will return an instance of OwlDialogRef. The OwlDialogRef provides a handle on the opened dialog. It can be used to close the dialog and to receive notification when the dialog has been open and closed.

When you subscribe dialogRef.afterClosed(), it receives a data object that has three parameters(action, originalEvent and value).

  • action: string -- It represents the type of action that closes the dialog. There are four types: 'confirm', 'cancel' any 'dismiss'. 'confirm' means the dialog is close when user clicked the confirm button. 'cancel' means the dialog is close when user clicked cancel button. 'close' means the dialog is close when user clicked close button. 'dismiss' means the dialog is close because this dialog has a timer and the timer is expired.
  • originalEvent: any -- It represents the original dom event that close the dialog.
  • value: any -- It represents the current value of the dialog. When dialog has input (text, textarea, radio, ect), the value is from the value of that input.

    public inputTypeChange( type: string ): void {
        switch (type) {
            case 'text':
            case 'textarea':
                this.dialogInputType = type;
                break;

            case 'select':
                this.dialogInputType = type;
                this.dialogInputPlaceholder = 'Select your favourite team';
                this.dialogInputValue = {id: 2, name: 'Los Angeles Lakers'};
                this.dialogInputOptions = {
                    'Golden State Warrior': {id: 1, name: 'Golden State Warrior'},
                    'Los Angeles Lakers': {id: 2, name: 'Los Angeles Lakers'},
                    'Cleveland Cavaliers': {id: 3, name: 'Cleveland Cavaliers'}
                };
                this.dialogInputCompareFn = ( ob1: any, ob2: any ): boolean => ob1.id === ob2.id;

                break;

            case 'radio':
                this.dialogInputType = type;
                this.dialogInputValue = 'Blue';
                this.dialogInputOptions = {
                    Red: 'Red',
                    Blue: 'Blue',
                    Green: 'Green'
                };

                break;

            case 'checkBox':
                this.dialogInputType = type;
                this.dialogInputValue = true;
                this.dialogInputLabel = 'I agree with the terms and conditions';

                break;

            default:
                this.dialogInputType = null;
                this.dialogInputValue = null;
        }
    }
