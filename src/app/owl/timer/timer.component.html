Description

Owl Timer is an Angular component to count down time. It consists of server flip boxes to show time with styles and animations.

Import

Getting Start

Owl Timer requires time value to count down. Use [timerDays], [timerHours], [timerMinutes] and [timerSeconds] to configure the countdown value (how many days, hours, minutes and seconds to count down to). The countdown would start from the moment when you start the timer, but you could override the start moment by setting property [startTime]. In addition, instead of setting the days, hours, minutes or seconds individually, you could set the end moment by setting property [endTime]. Owl Timer would calculate how many days, hours, minutes or seconds between the startTime and the endTime and set the countdown value.

Control the Timer

By default, the timer would not start automatically. User could set the property [autoStart] to true, and the timer would start automatically when the page is loaded.

In addition, user could also control the timer via OwlTimerComponent.

Timer Max Unit

By default, Owl Timer's max time unit is 'day'. It means it would show how many days, hours, minutes and seconds to count down. Even if the days, hours or minutes is 0, their timer flip boxes are still on the timer. User could change the max time unit by setting property [maxTimeUnit].

Properties for owl-timer

Name Type Optional Default Description
autoStart boolean optional false Defines whether the timer starts automatically.
timerDays number optional 0 Defines how many days to count down to.
timerHours number optional 0 Defines how many hours to count down to.
timerMinutes number optional 0 Defines how many minutes to count down to.
timerSeconds number optional 0 Defines how many seconds to count down to.
startTime Date optional null Defines the time the count down timer starts to count down from
endTime Date optional null Defines the time the count down timer counts down to
maxTimeUnit 'day' | 'hour' | 'minute' | 'second' optional 'day' Defines the max time unit.
locale
optional
Defines the locale

Methods from OwlTimerComponent

owl-timer is exposed as owlTimer.

Name Parameter Return
startTimer -- set the timer to start null null
pauseTimer -- set the timer to pause null null
resetTimer -- reset the timer null null
cancelTimer -- set the timer countDown value to 0 null null
addCDSeconds -- add certain seconds to the timer countDown value seconds: number -- seconds to add to the timer. It could be positive or negative. null
