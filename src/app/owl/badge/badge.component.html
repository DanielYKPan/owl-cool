Description

Owl Badge owl-badge is an Angular component, providing small status descriptors for UI elements. A badge consists of a small circle, typically containing a number or other short set of characters, that appears in proximity to another object. Owl Badge uses Angular animations. When its value changes, it would trigger a 'pulse' animation to notify users.

Import

Getting Start

Owl Badge should be placed inside a HTML element that serves as its parent. Its content could be provided via property [content].

Badge Position

Be default, the badge will be placed on the right, top corner of its parent element. The position could be changed by defining the property [positionY] as 'above' or 'below', defining the property [positionX] as 'left' or 'right'. Users could also adjust the badge position via its inline css style.

Badge Size

The badge has 3 sizes: 'small', 'medium' and 'large'. By default, the badge is set to 'medium'. Users can change the size via property [size]. Users could adjust the badge size via its inline css font-size style.

The badge visibility can be toggled programmatically by defining property [hidden].

Properties of owl-badge

Name Type Optional Default Description
description string Optional null Message used to describe the badge via aria-describedby
hidden boolean Optional false Define whether the badge is hidden.
content string require null Define the content of the badge. If the value changes, it would trigger a 'pules' animation.
positionX 'left' | 'right' optional 'right' Define position of the badge relative to its parent horizontally.
positionY 'above' | 'below' optional 'above' Define position of the badge relative to its parent vertically.
size 'small' | 'medium' | 'large' optional 'medium' Define size of the badge.
overlap boolean optional true Define whether the badge should be overlap its parent.
