Description

Owl Rating is an Angular component to allow users to get or set rating value. It is well designed, styled and animated.

Import

Getting Start

Owl Rating owl-rating requires rating buttons owl-rating-button. Users need to wrapper the rating buttons inside owl-rating. Each rating button needs to have a value representing the rating value. In addition, owl-chip-list is set up for both Template-driven Forms and Reactive Forms. User can manipulate the chip list's selected value using any of the form directives from the core FormsModule or ReactiveFormsModule: ngModel, formControl, etc.

By default, once the rating value choose, users could not remove the rating value. The could be changed by setting property [allowNull] to true.

Rating Icon Shape

By default, Owl Rating has star-shape icons. Users could change them to heart-shape icons by setting property [shape] to 'heart'. If user wants to have their own icons, use directive owl-rating-button-icon, and put it inside owl-rating-button.

Properties of owl-rating

Name Type Optional Default Description
allowNull boolean Optional false Defines whether to allow null rating value
disabled boolean Optional false When specified to true, the rating would be disabled.
name string Optional owl-rating-0 Defines a name for the rating button group. All rating buttons inside owl-rating will use this name.
shape 'star' | 'heart' Optional 'star' Defines the shape of the rating icons. Valid options are 'star' and 'heart'.
required boolean Optional false Defines whether the rating value is required.
readonly boolean Optional false Defines whether the attribute readonly of the rating button input.
tabIndex number Optional 0 Index of the element in tabbing order

Events of owl-rating

Name Parameters Description
ratingChange source: the rating component
originalEvent: event
value: ngModel rating value
 Event emits when rating value changed.

Properties of owl-rating-button

Name Type Optional Default Description
value number required null Defines the value of the rating button
aria-label string Optional null Attached to the aria-label attribute of the rating button
aria-labelledby string Optional null Users can specify the 'aria-labelledby' attribute which will be forwarded to the rating button
aria-describedby string Optional null Attached to the aria-describedby attribute of the rating button
shape 'star' | 'heart' Optional 'star' Defines the shape of the rating icons. Valid options are 'star' and 'heart'.
id string Optional 'owl-rating-button-0' Unique id for the rating button.

Events of owl-rating-button

Name Parameters Description
change source: the rating button component
originalEvent: event
value: the rating button value
 Event emits when rating button check state changed.
