Description

Owl Galleria is a component that allows user to display images and their content. It would animate the circular process of the images.

Import

Getting Start

Owl Galleria requires a set of images(OwlGalleriaImageOwlGalleriaImage) to display. Each image must include a src (the path of the image) and alt (the alt attribute of the image). It could also optionally includes frameSrc (the path of the image for filmStrip), desc(the summary of the image) and title(the title of the image). 



]]>

Film Strip

By default, Owl Galleria shows a film strip panel, indicating which image is active in the galleria. To disable the film strip panel, set property [showFilmStrip] to true.

Customise Galleria Panel Content

User could customise galleria panel's content by wrapping it inside owl-galleria. User could also use owlGalleriaNav directive to define a button as a galleria navigation to navigate the images in galleria.

Properties of owl-galleria

Name Type Optional Default Description
animationDuration number Optional 700 Defines the duration of galleria animation in milliseconds.
animationMode 'slide' | 'fade' Optional 'fade' Defines the mode of galleria animation. optional values: 'fade' and 'slide'.
autoPlay boolean Optional true Defines whether the image would change automatically in certain seconds
frameWidth string Optional '60px' Defines the width of the image in the galleria film strip.
frameHeight string Optional '40px' Defines the height of the image in the galleria film strip.
images OwlGalleriaImage[] required null Defines an array of images to display in the galleria
panelWidth string optional '600px' Defines the width of the galleria panel
panelHeight string optional '400px' Defines the height of the galleria panel
showFilmStrip boolean optional true Defines whether to display a film strip in galleria
transitionInterval number optional 4000 Defines the time in milliseconds between each slide in autoPlay mode
id string optional owl-galleria-0 Set the unique id for the galleria.
activeImage number optional 0 Define the current active image index.

Events of owl-galleria

Name Parameters Description
activeImageChange number -- the index number of the image in the array of the images. Event emits when the galleria's active image is changed.
imageClick originalEvent: event
value: number -- the index number of the image that is clicked. 		Event emits when the galleria's active image is clicked.

Properties of owlGalleriaNav

Name Type Optional Default Description
owlGalleriaNav OwlGalleriaComponent required null Defines which galleria panel this owlGalleriaNav is relative to.
prev boolean optional false Defines whether this owlGalleriaNav is used to change active image to previous one.
disabled boolean Optional false When specified to true, the owlGalleriaNav would be disabled.
aria-label string Optional null Attached to the aria-label attribute of the owlGalleriaNav
aria-labelledby string Optional null Users can specify the 'aria-labelledby' attribute which will be forwarded to the owlGalleriaNav

Properties of OwlGalleriaImage

Name Type Optional Default Description
src string required null Defines the path of the image in galleria
alt string required null Defines the alt attribute of the image in galleria
frameSrc string optional null Defines the path of the image in galleria's film strip, if it is null, the src of the image would be used in the film strip
desc string optional null Defines the summary of the image in galleria
title string optional null Defines the title of the image in galleria
