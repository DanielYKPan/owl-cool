Description

Owl Select is an Angular component for selecting a value / multiplie values from a set of options. It is designed to work like a native HTML element ]]>. It works inside of an ]]> element, having floating placeholder and underline animation.

Import

Getting Start

Owl Select requires a value to bind and a collection of options.

Options could be wrap inside owl-option, and use property [value] to define the option's value. User could also use it with ngFor to iterate through an array of objects. Note that user can disable certain options by adding the disabled boolean attribute or binding to it.
