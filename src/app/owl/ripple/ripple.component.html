Description

It is an Angular directive to make element have material ripple effects. Inspired by Angular Material Design, this implementation contains the ripple animation for buttons and links.

If the ripple directive's host element has a 'static' css position, it would make its position to be 'relative'. If this would affect your design, you could wrap the element into a 'div' block, and set your desire css position value to that 'div' block.

Animation

By setting property [owlRippleFadeInDuration], user could slow down or speed up the ripple spread process. The default value is 600ms.

By setting property [owlRippleFadeOutDuration], user could slow down or speed up the ripple disappear process. The default value is 400ms.

Customise Ripple Color and Radius

User could set property [owlRippleColor] to change the ripple's color. The default color is 'rgba(0, 0, 0, .1)'.

By default, the ripple effect is not visible outside the host element. User could override this by setting property [owlRippleUnbounded] to true.

Besides, the radius will be the distance from the center of the ripple to the furthest corner of the host element's bounding rectangle. Set [owlRippleRadius] to a value, which would define the radius in pixels of foreground ripples when fully expanded.

Trigger Element

Properties

Name Type Optional Default Description
owlRippleDisabled boolean Optional false When specified to true, click events will not trigger the ripple.
owlRippleCentered boolean Optional false When specified to true, the ripple would always start from the center of the element.
owlRippleFadeInDuration number Optional 600 The duration of ripple to fade in. The unit is 'ms'.
owlRippleFadeOutDuration number Optional 400 The duration of ripple to fade out. The unit is 'ms'.
owlRippleRadius number Optional null The radius of the ripple in 'px'.
owlRippleColor string Optional rgba(0, 0, 0, .1) The color of the ripple.
owlRippleClass string Optional null Class that is assign to ripple element.
owlRippleTriggerElement HTMLElement Optional ripple directive's host element Define the element that triggers the ripple when click events are received.
owlRippleUnbounded boolean Optional false Whether foreground ripples should be visible outside the ripple directive's host element bounds
