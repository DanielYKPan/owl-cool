Description

Owl Radio Button is an Angular component provides the same functionality as a native HTML input radio element ]]>. It is generally used in radio group owl-radio-group -- a collection of radio buttons owl-radio-button describing a set of related options. Only one radio button in a given group can be selected at a time.

Import

Getting Start

Users wrap their radio button's label between owl-radio-button, and use property [value] to set the radio button's value.

Several radio buttons need to be wrapped inside owl-radio-group, and use [name] to define the name of the radiogroup. In addition, owl-radio-group is set up for both Template-driven Forms and Reactive Forms. User can manipulate the select's value using any of the form directives from the core FormsModule or ReactiveFormsModule: ngModel, formControl, etc.

Properties of owl-radio-group

Name Type Optional Default Description
disabled boolean Optional false When specified to true, the radio group would be disabled.
labelPosition 'after' | 'before' Optional 'after' Defines the label position should be 'before' or 'after' the radio button.
name string Optional owl-radio-group-0 Defines a name for the radio button group. All radio buttons inside group will use this name.
required boolean Optional false Whether the radio group is required.
tabIndex number Optional null Index of the element in tabbing order
value any Require null The value assigned to the radio input.

Name Parameters Description
