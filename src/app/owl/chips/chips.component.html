Description

Owl Chip is a Angular component showing a chip-like individual. User could group several chips into a list, or add/remove them with a chip input.

Import

Getting Start

Single Chip

Wrap the chip text into owl-chip to show a single chip.

User could style a single chip by using inline style or css style class. In addition, individual chips may be disabled by applying the [disabled] attribute to the chip. When disabled, chips are neither selectable nor focusable.

Select Chip

owl-chip-list is set up for both Template-driven Forms and Reactive Forms. User can manipulate the chip list's selected value using any of the form directives from the core FormsModule or ReactiveFormsModule: ngModel, formControl, etc.

Chips can be selected by clicking SPACE when they are on focus. Users can also manually calling toggleSelected on owl-chip. 

My Chip
]]>

Keyboard Interaction

User could use arrow keys to navigate among chips. Use SPACE to select/deselect them, and use DELETE/BACKSPACE to fire callback function (remove) to get a removable chip reference. Chips also gain focus when clicked, ensuring keyboard navigation starts at the appropriate chip.
