Description

Owl Slider is an Angular component for selecting a value from an range input. User could control a slider via a mouse move, touch or keyboard. It is similar to ]]>.

Note: the sliding behavior for this component requires that HammerJS is loaded on the page. You could learn more about this from here.

Import

Getting Start

Owl Slider is set up for both Template-driven Forms and Reactive Forms. User can manipulate the slider's value using any of the form directives from the core FormsModule or ReactiveFormsModule: ngModel, formControl, etc. 


]]>

Owl Slider could increment or decrement value by 1 step at a time. Step value is 1 by default and user could change it via [step]. 


]]>

Max & Min

By default, Owl Slider has 100 as max value, and 0 as min. User could change those via [max] and [min]. 


]]>

Range Selection

Owl Slider could select a range value. User could set [range] to true, and provide a array with two values in it(index 0 for low value, index 1 for high value) to ngModel; 


]]>

Slider Direction

Owl Slider is in horizontal mode by default. Change [vertical] to true to set the slider as vertical mode. 


]]>

Balloon Label

By default, user could not see the selected value via the slider is sliding. This behaviour could be changed via setting [showBalloonLabel] to true to show a balloon label beside the slide handle. 


]]>

Tick Marks

By default, slider does not show tick marks on the slider track. This can enable by setting [tickInterval]. The value of tickInterval should be a number representing the number of steps between tow tick marks. For example, a tickInterval of 4 with a step of 5 will draw tick marks by every 4 steps, which is the same as 20 values. 


]]>

Keyboard Interaction

  • RIGHT_ARROW / UP_ARROW: Increment the slider value by one step.
  • LEFT_ARROW / DOWN_ARROW: Decrement the slider value by one step.
  • PAGE_UP: Increment the slider value by 10 step.
  • PAGE_DOWN: Decrement the slider value by 10 step.
  • HOME: Set the value to the maximum possible.
  • END: Set the value to the minimum possible.
  • SPACE: Toggle active slider handle in range mode.

Properties for owl-slider

Name Type Optional Default Description
aria-label string Optional null Attached to the aria-label attribute of the slider
aria-labelledby string Optional null Users can specify the 'aria-labelledby' attribute which will be forwarded to the slider
animate boolean Optional true When specified to false, the slider would NOT be animated.
disabled boolean Optional false When specified to true, the slider would be disabled.
showBalloonLabel boolean Optional false When specified to true, slider would have a balloon label beside the slider handle.
max number Optional 100 Defines the maximum value of the slider
min number Optional 0 Defines the minimum value of the slider
range boolean Optional false When specified to true, the slider would be in range mode.
step number Optional 1 Defines the factor to increment/decrement the value
tabIndex number Optional null Index of the element in tabbing order
tickInterval number Optional 0 Defines how often to show ticks mark in slider track.
vertical 'horizontal' | 'vertical' Optional 'horizontal' Defines the direction of the slider.
